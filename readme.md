# Dashboard
Goals:
    This is a dashboard with CMS.
    user doesnt require login
    admin must login to update the dashboard data; may also upload the csv/xlsx file;
    backend will update the database.

Required features:
    user login management.
    csv/excel parsing to update the database.
    charts display; ability to add/remove widgets
    changable themes.
    
Technologies to be used:
    bootstrap5
    HTMX/apline.js
    golang
    sqlite3
